# Xdownmod Public Frontend Clone

هذه الحزمة ليست ملفات B12 الأصلية الداخلية.
هي نسخة عامة مُعاد بناؤها اعتمادًا على المحتوى العام الظاهر على الموقع وروابط الأصول العامة المتاحة.

## الموجود
- index.html
- assets/css/style.css
- assets/js/app.js

## ملاحظة
لا يمكن استخراج ملفات الإدارة الخلفية أو ملفات البناء الخاصة بمنصة B12 من الواجهة العامة.
